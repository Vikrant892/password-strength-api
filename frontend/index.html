PWANALYZE(1) User Commands PWANALYZE(1) NAME pwanalyze — entropy and breach analysis for passphrases SYNOPSIS pwanalyze [--length] [--charset] [--hibp-check] DESCRIPTION Computes Shannon entropy and classical crack-time estimates for a given passphrase. Pattern checks catch common substitutions, keyboard walks, and rockyou-style weak passwords. Optional HIBP check uses k-anonymity so the passphrase never leaves the client.

  client                                    haveibeenpwned.com
    │                                             │
    │  SHA-1(password) = AABBCCDDEE...             │
    │                                             │
    │  GET /range/AABBC  ────────────────────────→ │
    │                                             │
    │  ←─────────── 500 hashes starting AABBC      │
    │                                             │
    │  check if DDEE... in response                │
    │  (password itself never leaves client)       │

  tool sends only the first 5 hex chars of the sha-1 hash.
  server returns ~500 matching suffixes.
  client decides locally whether the hash appears.
