k-Anonymity: How it works

The CryptoGuard API checks passwords against the Have I Been Pwned database without ever sending the full password over the network.

  1. The password is hashed client-side using SHA-1.
  2. Only the first 5 characters of the hash are sent to the HIBP API.
  3. HIBP returns all hash suffixes matching that prefix (typically 500-800 results).
  4. The full hash is compared locally against the returned list.
  5. The server never sees the complete hash, preserving full privacy.
